# Помощники для разработчиков

Этот набор помогает разработчику или техническому специалисту проектировать изменения, отлаживать проблемы, проверять код и работать с современным веб-стеком.

## Что доступно

- Мозговой штурм и планирование: Claude помогает уточнить задачу, спроектировать решение и составить план реализации.
- Разработка через тесты: Claude помогает сначала определить проверяемое поведение, а затем писать код.
- Системная отладка: Claude помогает идти от симптома к причине, проверять гипотезы и не лечить проблему наугад.
- Code review: Claude помогает подготовить код к ревью, принять замечания и проверить риски регрессий.
- Git worktrees и завершение ветки: Claude помогает аккуратно работать с ветками, проверками и финальным состоянием перед PR.
- Next.js, Vercel, shadcn, Supabase: Claude помогает с задачами по веб-приложениям, UI-компонентам, деплою и PostgreSQL/Supabase-практикам.
- WordPress-блог ISB: Claude помогает с задачами по блогу, SEO, темам и публикациям при наличии доступа к нужным инструментам.

## Как спросить

- "Помоги отладить эту ошибку системно."
- "Составь план реализации и проверки."
- "Проверь мой код как ревьюер."
- "Какие AI-скиллы доступны для разработчика?"
