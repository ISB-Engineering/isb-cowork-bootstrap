# Сервис

Этот набор помогает сервису структурировать обращения, готовить отчеты и планировать следующие действия по заявкам.

## Что доступно

- Мозговой штурм по обращению: Claude помогает понять возможные причины, вопросы для уточнения и варианты решения.
- Планирование работ: Claude помогает оформить шаги диагностики, выезда или follow-up.
- Проверка результата: Claude помогает убедиться, что отчет понятен и задачу можно закрывать.

## Как спросить

- "Помоги разобрать обращение клиента."
- "Составь план диагностики."
- "Проверь сервисный отчет перед отправкой."
- "Какие AI-скиллы доступны для сервиса?"
