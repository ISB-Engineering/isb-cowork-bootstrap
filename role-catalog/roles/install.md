# Монтаж

Этот набор помогает монтажу быстрее фиксировать ситуацию, готовить планы работ и не терять важные детали перед завершением задачи.

## Что доступно

- Мозговой штурм по проблеме на объекте: Claude помогает разложить ситуацию и найти варианты действий.
- План работ: Claude помогает оформить последовательность действий, вопросы, риски и контрольные точки.
- Проверка результата: Claude помогает сверить, что задача действительно закрыта и есть факты для отчета.

## Как спросить

- "Помоги составить план работ на объекте."
- "Разложи проблему и варианты решения."
- "Проверь, чего не хватает в отчете."
- "Какие AI-скиллы доступны для монтажа?"
